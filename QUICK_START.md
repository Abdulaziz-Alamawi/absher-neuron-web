# دليل البدء السريع

## خطوات التشغيل

### 1. تثبيت الحزم
```bash
npm install
```

### 2. تشغيل خادم التطوير
```bash
npm run dev
```

### 3. فتح المتصفح
افتح المتصفح على العنوان:
```
http://localhost:3000
```

## الصفحات المتاحة

- **الصفحة الرئيسية**: `http://localhost:3000/`
- **لوحة تحكم المستخدم**: `http://localhost:3000/dashboard`
- **لوحة تحكم محلل الأمن**: `http://localhost:3000/security`
- **صفحة حول النظام**: `http://localhost:3000/about`
- **تفاصيل التنبيه**: `http://localhost:3000/alerts/ALR-0001`

## ملاحظات

- جميع البيانات حالياً هي بيانات وهمية (Mock Data) للتطوير
- النظام يدعم RTL بالكامل
- التصميم متجاوب ويعمل على الجوال والكمبيوتر
- الألوان مستوحاة من منصة أبشر (أخضر/أبيض)

## البناء للإنتاج

```bash
npm run build
npm start
```
